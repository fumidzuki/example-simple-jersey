# simple-webapp-gradle

## Overview

Java Servlet を Gradle だけで実行するためのサンプルプロジェクトです。

## Requirement

* Java Development Kit（JDK）：11+

## Install

```sh
$ git clone https://github.com/fumidzuki/example-simple-servlet.git
```

## Usage

```sh
$ cd example-simple-servlet
$ gradlew appRun
```
